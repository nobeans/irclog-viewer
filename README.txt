What's this?
------------

This is a simple viewer of IRC log archives.  It's providing serivce only to view.
The acumuration is not out of services.  To import log you have to construct others way.


Getting start
-------------

To try it, clone this repository from github and execute the following command:

  $ grails irclog-import-dummydata
  $ grails run-app

so the application will be started, and you can access to http://localhost:8080/irclog.


To import log
-------------

To import IRC log into the database of this application, 
for example, you can use simple bot powered by groovy.
